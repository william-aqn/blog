Как подключиться

Адрес сервера
  1. Открой Minecraft Java (версия 26.1.2)
  2. «Сетевая игра» → «Добавить сервер»
  3. Вставь адрес: {{ site.minecraft_address }}
  4. Жми «Подключиться»
А пароль?

При входе само появится окошко с паролем — в чат ничего писать не надо.

1-й разпридумай пароль и повтори его. Обязательно запомни!
